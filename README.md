# ShoppyGlobe_Backend

Node.js + Express + MongoDB E-commerce Backend

GitHub Link : <https://github.com/raghuwanshiayush2004/ShoppyGlobe_backend>

## Features

- Product CRUD
- Shopping Cart
- JWT Auth
- MongoDB

## Setup

```bash
npm init
npm i express mongoose nodemon
npm install bcryptjs jsonwebtoken dotenv cors

Run Server
npm start
